Community Science Museum

Link to Github Link to deployed site

What was changed?

  1. Some simple code cleanup.
  2. Fixed headings to make it hierarchical
  3. Changed the scroll when overflow in 'Exhibition spaces' page
  4. In the home page section 'Events and News'i made the entire image an anchor tag so it is all clickable

How I changed it

  1. Fixed some tags and file names that was uppercase and deleted the unecessary @font-face from the stylesheet. I did also place the stylesheet link in the correct order in each html document. Image files where edites to have the accurate formatting without spaces
  2. Changed headings in almost all pages
  3. To delete the scroll and overflow i had to change the layout of the entire page. There was some position issues that made some section overflow over others. The use of relative and absolute position for the section made it difficult to fix the page so i had to place the images over the text section without it being in position absolute
  4. By using flexbox on the section and each anchor tag. This made it also more responsives

Why I did it that way

  1. Small code error can affect the entire page and create bugs. It can also make the code be bigger in size than necessary
  2. After understanding how the heading should work, i had to take time to change most of them from previous work. This hierarchy makes it more accessible when specially using assistive technology by making the page leveled or ranked by importance
  3. The overflow scroll made the page cluttered and messy. It wasnt very cleared that there was something more to read so changing it makes it more possible for the user to read all the paragraph
  4. Easier to access the link by making the entire image an anchor tag
